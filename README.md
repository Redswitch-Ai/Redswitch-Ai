## 🚀 Quick Start

`ash
pip install redswitch
`

`python
from redswitch import RedSwitch

rs = RedSwitch(
    agent_id='my-agent',
    human_id='you@email.com',
    platform='openclaw'
)

rs.register()
rs.heartbeat()  # Call this periodically
`

- **Multi-Channel Alerts** — Email, Telegram, SMS
